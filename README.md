# Smart Table

Приложение для отображения, фильтрации, сортировки и пагинации данных о продажах в виде интерактивной таблицы.

## 📋 Описание

Smart Table — это SPA-приложение на ванильном JavaScript, которое предоставляет удобный интерфейс для работы с данными о транзакциях. Приложение загружает данные с удалённого API и позволяет:

- 🔍 **Поиск** — глобальный поиск по всем полям таблицы
- 🎯 **Фильтрация** — фильтрация по дате, покупателю, продавцу и сумме транзакции
- ↕️ **Сортировка** — сортировка по дате и сумме (по возрастанию/убыванию)
- 📄 **Пагинация** — постраничная навигация с настраиваемым количеством записей на страницу

## 🛠 Технологии

- **Vite** — сборщик проектов
- **Vanilla JavaScript (ES6+)** — без фреймворков
- **CSS Custom Properties** — кастомизация стилей через CSS-переменные
- **Fetch API** — работа с сервером

## 📡 API

Приложение работает с удалённым API:

```
BASE_URL: https://webinars.webdev.education-services.ru/sp7-api
```

### Эндпоинты

| Метод | Эндпоинт     | Описание                        |
| ----- | ------------ | ------------------------------- |
| GET   | `/records`   | Получение записей о транзакциях |
| GET   | `/sellers`   | Получение списка продавцов      |
| GET   | `/customers` | Получение списка покупателей    |

### Параметры запроса

- `filter[date]` — фильтрация по дате
- `filter[customer]` — фильтрация по покупателю
- `filter[seller]` — фильтрация по продавцу
- `filter[totalFrom]` — минимальная сумма
- `filter[totalTo]` — максимальная сумма
- `sort` — сортировка (поле и направление)
- `page` — номер страницы
- `rowsPerPage` — количество записей на страницу

## 🧩 Компоненты

### Таблица (`table.js`)

Основной компонент для рендеринга таблицы. Использует шаблоны из `index.html`:

- `#table` — структура таблицы
- `#row` — шаблон строки данных
- `#header` — шапка таблицы
- `#filter` — строка фильтров
- `#pagination` — пагинация

### Фильтрация (`filtering.js`)

Функция `initFiltering()` возвращает методы:

- `updateIndexes(elements, indexes)` — заполнение dropdown-списков (например, список продавцов)
- `applyFiltering(query, state, action)` — применение фильтров к запросу

### Сортировка (`sorting.js`)

Функция `initSorting()` управляет визуальным состоянием кнопок сортировки (по возрастанию/убыванию/без сортировки).

### Поиск (`searching.js`)

Функция `initSearching()` обрабатывает глобальный поиск по таблице.

### Пагинация (`pagination.js`)

Функция `initPagination()` управляет:

- Отображением кнопок страниц
- Навигацией (первая/предыдущая/следующая/последняя страница)
- Статусом («Showing X to Y of Z entries»)
- Выбором количества строк на страницу (10/25/50/100)
